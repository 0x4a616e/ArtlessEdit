    ArtlessEdit

    An artless editor for OS X bringing Ace to your desktop.

    ArtlessEdit allows you to user the beautiful Ace editor with the look and feel of a native OS X application.

    In addition to the rich feature set of Ace, ArtlessEdit brings you:

    • Quickly search through your recently opened files
    • Browse your recent file versions using eclipse-like fuzzy matching
    • Configure the look and feel separately by language
    • Navigate your files using exuberant ctags
    • Paste from your pasteboard history

    For a complete list of features, check out ace.
